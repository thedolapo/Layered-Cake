# Layered Cake - Three.js

A beautiful 3D layered cake visualization built with Three.js. This interactive cake features multiple layers, decorative frosting drips, and a Victorian-style stand.

## About

I was working on a project and needed a 3D cake rendered in a canvas on the web. This is the result - a fully interactive, beautifully textured layered cake built with Three.js that can be embedded anywhere on the web.

## How to Run

Simply open `index.html` in a modern web browser. The project uses ES modules and imports Three.js from a CDN, so no build process is required.

## Technologies

- **Three.js** (v0.152.2) - 3D graphics library
- **OrbitControls** - Camera controls for interaction
- **Vanilla JavaScript** - No frameworks required

## Controls

- **Left Click + Drag** - Rotate the camera around the cake
- **Right Click + Drag** - Pan the camera
- **Scroll Wheel** - Zoom in/out

## Browser Compatibility

Works best in modern browsers that support:
- ES6 Modules
- WebGL
- Canvas API
